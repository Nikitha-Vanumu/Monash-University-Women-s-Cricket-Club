# Monash University Women's Cricket Club

A modern, full-featured website for the Monash University Women's Cricket Club — built to showcase the club's history, training programs, and season activity to members, prospective players, and the wider community.

🔗 **Live site:** [muwcc.netlify.app](https://muwcc.netlify.app/)

## Features

- Animated hero section with rotating imagery and calls to action
- Full club history timeline (1967–present), including season highlights and results
- Training program overview (schedule, drills, coaching, match prep)
- Live fixtures and results, integrated with external competition data
- Facebook photo feed integration for ongoing season content
- Sponsor showcase
- Fully responsive design across desktop and mobile

## Tech Stack

- React
- HTML / CSS / JavaScript
